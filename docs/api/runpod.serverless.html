runpod.serverless

package documentation
(source)

Contains the main entrypoint for the RunPod Serverless Worker.

Arguments can be passed in when the worker is started, and will be passed to the worker.

Module core Core functionality for the runpod serverless worker.
Package modules Allows modules to be imported from the modules directory.
Package utils Allows for the import of all modules in the utils directory.
